# Linux Update

## Recommended Commands

```bash
sudo apt update
sudo apt full-upgrade -y
sudo apt autoremove -y
sudo reboot
```

## What These Commands Do

### `sudo apt update`
Refreshes the package list and checks for available updates.

### `sudo apt full-upgrade -y`
Installs all available updates, including dependency changes if needed.

### `sudo apt autoremove -y`
Removes unnecessary packages and old dependencies.

### `sudo reboot`
Restarts the system after updates for best performance.

## Notes

- Keep your internet connection stable during updates.
- A reboot may be required after kernel updates.
- This process helps keep Kali Linux secure and up to date.
